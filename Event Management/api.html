API Documentation

Integrate with EventHub's powerful event management platform

Overview

The EventHub API allows developers to integrate EventHub's powerful event management capabilities into their own applications. With our API, you can create and manage events, handle registrations, process payments, and access analytics data programmatically.

Get API Key View on GitHub

Base URL

https://api.eventhub.com/v1

Authentication

All API requests must include your API key as a header. You can get your API key from the developer dashboard after signing up for an API account.

// Example request with API key
curl -X GET \
  -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.eventhub.com/v1/events

Endpoints

GET

/events

Returns a list of events. You can filter by various parameters such as date, category, and location.

Query Parameters:

  • category - Filter by event category
  • date_start - Filter by start date (ISO 8601 format)
  • date_end - Filter by end date (ISO 8601 format)
  • location - Filter by location
  • limit - Number of results per page (default: 20)
  • page - Page number (default: 1)

Example Response:

{
  "events": [
    {
      "id": "evt_123456",
      "title": "Web Development Summit",
      "description": "Join industry leaders...",
      "category": "Tech",
      "start_date": "2023-10-15T09:00:00Z",
      "end_date": "2023-10-17T17:00:00Z",
      "location": "New York City",
      "organizer_id": "org_789012",
      "status": "published",
      "created_at": "2023-07-01T12:00:00Z"
    },
    // More events...
  ],
  "meta": {
    "total_count": 42,
    "page": 1,
    "limit": 20
  }
}
POST

/events

Creates a new event with the specified parameters.

Request Body:

{
  "title": "Web Development Summit",
  "description": "Join industry leaders...",
  "category": "Tech",
  "start_date": "2023-10-15T09:00:00Z",
  "end_date": "2023-10-17T17:00:00Z",
  "location": "New York City",
  "capacity": 500,
  "is_public": true
}

Response:

Client Libraries

We provide official client libraries for several programming languages to make integration even easier.

JavaScript

npm install eventhub-js

Python

pip install eventhub-python

PHP

composer require eventhub/php

Webhooks

EventHub can send webhook notifications to your application when certain events occur. This allows you to build real-time integrations with our platform.

Available Webhook Events

  • event.created Triggered when a new event is created
  • event.updated Triggered when an event is updated
  • registration.new Triggered when an attendee registers for an event
  • payment.success Triggered when a payment is successfully processed
